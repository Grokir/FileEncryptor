# FileEncryptor
**So far, it encrypts  one  selected  file.**
Currently supported  algorithms: 
- [DES](https://en.wikipedia.org/wiki/Data_Encryption_Standard) 
- [DESX](https://en.wikipedia.org/wiki/DES-X) > Check reference in my [repo](https://github.com/Grokir/DES_and_DESX_algorithms).

In the future:
- AES 128, 192 and 256.
